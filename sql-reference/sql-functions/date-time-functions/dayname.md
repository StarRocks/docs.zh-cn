# dayname

## 功能

返回日期对应的日期名字

## 语法

```Haskell
DAYNAME(date)
```

## 参数说明

`date`: 支持的数据类型为 DATE 或 DATETIME 类型

## 返回值说明

返回值的数据类型为 VARCHAR 类型

## 示例

```Plain Text
MySQL > select dayname('2007-02-03 00:00:00');
+--------------------------------+
| dayname('2007-02-03 00:00:00') |
+--------------------------------+
| Saturday                       |
+--------------------------------+
```
