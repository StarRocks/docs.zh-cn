# dayofyear

## 功能

获取日期中对应当年中的哪一天

## 语法

```Haskell
DAYOFYEAR(date)
```

## 参数说明

`date`: 支持的数据类型为 DATE 或 DATETIME 类型

## 返回值说明

返回值的数据类型为 INT

## 示例

```Plain Text
MySQL > select dayofyear('2007-02-03 00:00:00');
+----------------------------------+
| dayofyear('2007-02-03 00:00:00') |
+----------------------------------+
|                               34 |
+----------------------------------+
```
