# to_date

## 功能

返回 DATETIME 类型中的日期部分

## 语法

```Haskell
TO_DATE(datetime)
```

## 参数说明

`datetime`: 支持的数据类型为 DATETIME

## 返回值说明

返回值的数据类型为 DATE

## 示例

```Plain Text
MySQL > select to_date("2020-02-02 00:00:00");
+--------------------------------+
| to_date('2020-02-02 00:00:00') |
+--------------------------------+
| 2020-02-02                     |
+--------------------------------+
```
