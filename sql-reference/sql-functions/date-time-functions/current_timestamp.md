# current_timestamp

## 功能

获取当前的时间

## 语法

```Haskell
CURRENT_TIMESTAMP()
```

## 参数说明

无

## 返回值说明

返回值的数据类型为 DATETIME

## 示例

```Plain Text
MySQL > select current_timestamp();
+---------------------+
| current_timestamp() |
+---------------------+
| 2022-03-21 15:32:53 |
+---------------------+
```

## 关键词

CURRENT_TIMESTAMP, CURRENT, TIMESTAMP
