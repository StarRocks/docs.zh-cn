# weekofyear

## 功能

获取一年中的第几周

## 语法

```Haskell
WEEKOFYEAR(date)
```

## 参数说明

`date`: 支持的数据类型为 DATE 或 DATETIME 类型  

## 返回值说明

返回值的数据类型为 INT

## 示例

```Plain Text
MySQL > select weekofyear('2008-02-20 00:00:00');
+-----------------------------------+
| weekofyear('2008-02-20 00:00:00') |
+-----------------------------------+
|                                 8 |
+-----------------------------------+
```
