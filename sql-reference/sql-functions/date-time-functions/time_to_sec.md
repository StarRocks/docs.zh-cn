# time_to_sec

## 功能

函数返回将参数 date 转换为秒数的时间值, 转换公式为: `(小时 * 3600 + 分钟 * 60 + 秒)`

## 语法

```Haskell
time_to_sec(date)
```

## 参数说明

`date`: 支持的数据类型为 DATETIME 类型

## 返回值说明

返回值的数据类型为 INT

## 示例

```plain text
mysql> select time_to_sec('12:13:14');
+-----------------------------+
| time_to_sec('12:13:14')     |
+-----------------------------+
|                        43994|
+-----------------------------+
```
