# year

## 功能

获取日期中的年的信息, 返回值范围为 1000-9999

## 语法

```Haskell
YEAR(date)
```

## 参数说明

`date`: 支持的数据类型为 DATE 或 DATETIME 类型  

## 返回值说明

返回值的数据类型为 INT

## 示例

```Plain Text
MySQL > select year('1987-01-01');
+-----------------------------+
| year('1987-01-01 00:00:00') |
+-----------------------------+
|                        1987 |
+-----------------------------+
```
