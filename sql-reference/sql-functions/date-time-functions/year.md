# year

## 功能

返回指定日期中的年份。

## 语法

```Haskell
INT YEAR(DATETIME date)
```

## 参数说明

`date`: 必填。支持DATE或DATETIME类型。

## 返回值说明

返回INT类型的值，范围1000-9999。
如果`date`的数据类型非DATE或DATETIME，或日期格式错误，返回NULL。如果`date`值为NULL, 返回NULL。

## 示例

```Plain Text
MySQL > select year('1987-01-01');
+-----------------------------+
| year('1987-01-01 00:00:00') |
+-----------------------------+
|                        1987 |
+-----------------------------+
```
