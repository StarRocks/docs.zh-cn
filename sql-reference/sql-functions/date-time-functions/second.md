# second

## 功能

获取日期中的秒的信息, 返回值范围为 0-59

## 语法

```Haskell
SECOND(date)
```

## 参数说明

`date`: 支持的数据类型为 DATE 或 DATETIME 类型

## 返回值说明

返回值的数据类型为 INT

## 示例

```Plain Text
MySQL > select second('2018-12-31 23:59:59');
+-----------------------------+
|second('2018-12-31 23:59:59')|
+-----------------------------+
|                          59 |
+-----------------------------+
```
